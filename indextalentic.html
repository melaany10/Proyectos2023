Proyecto Talentic

¿QUE ES TALENTIC?

Talentic es un repositorio de sistemas de información documental de los proyectos formativos realizados por los aprendices, en el ámbito de contenidos digitales. Este ofrece información a personal interno o externo del centro de formación, aplicando así también para las masas sectoriales y el sector empresarial.

¿Qué mas podemos saber del proyecto N° 1?

Conoceremos más a fondo la estructura de los proyectos; mirando cuáles fueron sus bases, objetivos, específicos y como fue ese proceso de creación para llegar a un resultado exitoso, tanto como la maquetación de la página web hasta sus ideales.

¿CUÁL ES EL OBJETIVO PRINCIPAL DEL PROYECTO?

El objetivo principal de este proyecto es poder brindar a las personas que visitan este sitio web y a los aprendices información de proyectos ya realizados por los aprendices.

Como otro objetivo principal es visibilizar las habilidades blandas y técnicas de cada uno de los aprendices a nivel de conocimientos multimedia y de contenidos digitales.

¿CÓMO ESTA CONSTITUIDO EL PROYECTO (origen)?

La idea del proyecto se lleva trabajando aproximadamente hace 3 años, siendo aprobado y apoyado por SENNOVA.

¿QUÉ HERRAMIENTAS SE UTILIZARON PARA DESARROLLAR LA PÁGINA WEB?

La página web está constituida o conformada:

- HTML5, sobre un Frame bootstrap 5

- Flexbox Grid

- PHP, para el desarrollo de programación y de almacenamiento de datos teniendo en cuenta un servidor web.
