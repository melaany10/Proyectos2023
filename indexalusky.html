Proyecto Alusky

¿CUAL ES EL OBJETIVO PRINCIPAL DEL PRYECTO?

El objetivo principal del proyecto "Alusky", es dar a conocer y exponer los productos realizados por los aprendices, que en este caso prendas de vestir con sublimación.

¿Qué mas podemos saber del proyecto N° 5?

Conoceremos más a fondo la estructura de los proyectos; mirando cuáles fueron sus bases, objetivos, específicos y como fue ese proceso de creación para llegar a un resultado exitoso, tanto como la maquetación de la página web hasta sus ideales.

¿CÓMO ESTA CONSTITUIDO EL PROYECTO (origen)?

Como tal, lo que ellos más se centraron fue en la página web; dentro de ellas se encuentran diferentes funciones y una de las que más resaltaron fue la recopilación de información de todos los datos de las personas que se registraban en la página.

Estaba constituida por dientes archivos o programas que ayudaban al funcionamiento de la página. En el grupo eran dos integrantes.

¿QUÉ HERRAMIENTAS SE UTILIZARON PARA DESARROLLAR LA PÁGINA WEB?

Las herramientas que utilizaron para realizar la página web, fueron:

- Visual Studio Code - HTML5 - CSS - Python - JavaScript - Una base de datos, mediante carpetas categorizadas.
